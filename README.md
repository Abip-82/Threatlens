# Threatlens
Threatlens is a simple educational webapp that contains various cybersecurity toolkits to help users with various security threats. The initial version include a simple URLs analyzer, that analyzes suspicious links entered by user and determines potential phishing risks by calculating risk score. However, later versions WILL contain more features like hash generation, password strength calculator, cipher text (encoder for plain texts and decoder for cipher texts), and much more.

Note: This web app is  aimed to be purely educational and for learning purposes. The results obtained from here may or maynot be final/legit. Therefore it is important to check and validate resuls before drawing any conclusions.

Tech Stacks used:
Backend : Python (Flask backend)
Frontend : Javascript 

How it works?
User input is collected (which will be a URL in initial versions but on later ones, it might be hash or passwords, and others), which is then processed and cleaned. That data is then analyzed through various detection mechanisms and suspicious things are flagged. Results are then displayed to the user.

Uses:
This can be used by various students (or learnes in general) to analyze and detect and be aware of suspicious links. Upcmoning features like hash generation, cipher encoder/decoder will be useful for CTF players mostly. Common features like password strength generator can be used by everyone!
